Temperature Sensor

This is a personal Arduino project I created using an UNO board, a DHT11 sensor, and a 4-pin I2C LCD display. The goal was to build a reliable system that displays real-time temperature and humidity values directly from environmental readings.

I used this setup to identify areas in my home where the air conditioning and heating seemed to be the most inefficient. I placed the sensor in different rooms and observed the output over time.

After logging data, I found that the top main bedroom and the far bedroom were the least effective in terms of airflow and temperature balance. On the other hand, the powder bathroom had the most consistent output, likely because it‚Äôs closest to the AC unit and main vent connections.

This project taught me how hardware can help identify practical energy inefficiencies, and how sensors can guide real-world optimizations in comfort and energy usage.

Joystick Controlled Tetris Replica

This Tetris replica was built using an Arduino UNO, an LED matrix, and a 2-axis joystick as a controller.

I wanted to challenge myself to recreate a playable version of a classic arcade game, but with physical controls instead of a screen interface. I programmed the falling blocks, collision detection, row clearing, and game-over logic entirely in Arduino C++.

The joystick is used to move and rotate the blocks, and the LED matrix updates the display in real time. Later, I experimented with sharing scores online through a serial link to display rankings.

This project taught me about handling timing loops, hardware synchronization, and interactive design. I learned how important it is to plan the game logic, optimize the refresh rate, and make it feel responsive.
